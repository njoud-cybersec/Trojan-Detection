# Trojan Detection using Machine Learning

## Description
Machine Learning project to detect Trojan software using SVM and Logistic Regression.

## Libraries
- pandas, numpy
- matplotlib, seaborn
- scikit-learn

## Features
- Data cleaning and preprocessing
- Missing value handling
- Feature selection using SelectKBest
- Model training: SVM (RBF kernel) and Logistic Regression
- Model evaluation with confusion matrix and classification report

## How to run
1. Load dataset CSV
2. Run `main.py` or notebook
3. Check results and confusion matrices
